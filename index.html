🗺️ 产业链

📰 简报

每日最重要的10条消息 + 10只精选标的。由DeepSeek V4-Pro基于实时行情+真实新闻生成，每小时更新。每条消息含原文链接。标绿色为当前推荐，点代码跳通达信。

📡 快讯

实时滚动新闻流。每5分钟从新浪财经+东财公告抓取。分两路：赛道新闻（蓝点，与63赛道相关）+ 市场宏观（黄点，政策/外围/大宗）。点击标题跳原文。

📅 事件日历

未来催化事件按日期排列。⚡关键=硬催化（停产/涨价/法规/财报）。红色边框=重大事件。已过期自动折叠。所有事件含来源链接。

📋 布局

根据事件日历自动生成布局卡片。倒计时天数（红色<3天 橙色<7天 蓝色<14天）。每张卡片含赛道+建议提前天数+产业链标的。已过期自动折叠到历史归档。

🎯 标的

每个赛道6只精选核心标的（固定维护，非实时数据）。主板优先，科创/创业板最多3只。覆盖63赛道×6只=378只精选个股。点击代码跳通达信。

🔧 手动刷新手把手教学

什么时候用：上班打开网站发现数据没更新，点右上角🔄按钮。GitHub Actions cron偶尔不触发，这是保底手段。

第一次使用（30秒配置，一劳永逸）：

1. 打开 github.com/settings/tokens

2. 点 Generate new token (classic)

3. 勾选 workflow 权限（其他全部不勾）

4. 复制生成的 token（ghp_开头），粘贴到浏览器弹窗

5. 以后点🔄按钮就能直接触发更新，永远不需要再输 token

工作原理：按钮 → 调用GitHub API → 触发 market-update.yml 在GitHub云端执行 → 拉行情写data.json → GitHub Pages自动部署 → 网站刷新。全程约90秒。

安全说明：Token 仅存在你浏览器本地（localStorage），不会上传到任何服务器。每次调用直接访问 api.github.com，没有中间人。
